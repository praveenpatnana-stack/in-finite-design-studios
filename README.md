IN FINITE DESIGN STUDIOS :

--> In Finite Design Studios is a contemporary architecture and interior design portfolio website showcasing residential, hospitality, healthcare, and institutional projects. The platform reflects a refined design philosophy centered around spatial elegance, material expression, and experiential architecture.

About the website :
1. Showcase architectural projects with visual storytelling
2. Present design philosophy through refined layouts
3. Provide an immersive project viewing experience
4. Enable smooth navigation across works and studio information
5. Offer a responsive, performance-optimized experience across devices

Featured sections :
1. Works — Residential, Hospitality, Healthcare & Institutional Projects
2.  The Collective — Studio team members
3. Services — Design expertise & offerings
4. Contact — Studio details & project inquiry form

Build with :
1. React
2. TypeScript
3. Vite
4. Tailwind CSS
5. Framer Motion

Development :
1. Install dependencies --> npm install
2. Run locally --> npm run dev
3. Build for production --> npm run build

Deployment :
--> The site is deployed via Netlify with automatic CI/CD from GitHub
